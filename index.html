Guia para o Usuário

Percussionista, ao abrir o app ele já está configurado para: tocar em andamento mais lento (-30 BPM); iniciar com contagem de 2 compassos; metrônomo ligado; loop ativado (pratique várias vezes sem soltar as baquetas); reproduzir todos os instrumentos juntos. Os 4 staffs da partitura soam como: 1º — Oboé, 2º — Triângulo, 3º — Caixa e 4º — Bumbo Sinfônico. Se estiver com atraso no Bluetooth, use o botão específico para calibrar.

Para visualizar melhor a partitura, clique em "Foco".

Para abrir uma nova música: use o botão de pasta ou selecione pelo menu HNC Bb.
